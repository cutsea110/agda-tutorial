Mutual definitions

To allow mutual definitions one should declare any set before using it:


data L : Set
data M : Set

data L where
  nil : L
  _∷_ : ℕ → M → L

data M where
  _∷_ : Bool → L → M

Note that : Set is missing in the definitions of sets declared before.
